Packagecom.specialmoves.listComponent.eventManagement
Classpublic final class EventManager
InheritanceEventManager Object
Implements IEventManager

Class to allow the use of events and automativclly switch between mouse and touch events based on availability Will speed up and prevent bubbling, uses signals instead to dispatch to registered handlers Automatically switches to MultitouchInputMode.TOUCH_POINT if available
